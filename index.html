FontUnstack Example

The title on this page uses the “Zapfino” as the first choice. This typeface has very extreme metrics and really benefits from some custom styling, but only when it is present. In this example, when Zapfino is installed we use a negative bottom margin, a smaller size, and a slight negative left margin, to bring get it to align with this paragraph. If you don't have Zapfino, you get a more standard typeface (Gill Sans, Helvetica, or Arial), and we don't need these styles.

The body text uses Garamond (preferred), then Georgia and Times/Times New Roman. Garamond has a small x-height so needs less leading, Georgia has a much bigger x-height making it appear much larger, so benefits from more lead.

“A hanging pullquote…”

A hanging quote (when the “ sits outside the left margin) is normally difficult to achieve using relative type sizes when you don't actually know what typeface is used. In the above example, Garamond, Georgia, and Times/Times New Roman all need a different negative margin (-.40em, -.35em, and -.45em respectively. )
